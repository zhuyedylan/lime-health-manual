# 注册指南

![注册](/img/0注册-1.png)

欢迎使用柠健康（LiMe）APP！

为帮助您顺利创建账号并配置个人档案，请按照以下步骤操作：

## 第一步：本机号码一键登录与注册

1. 打开 APP 后，进入免密码登录和注册页面
2. 系统会自动识别并掩码显示您的本机手机号码（例如：188****8888）
3. 勾选下方确认圆圈，代表您已阅读并同意《用户服务协议》、《隐私政策》和《运营商协议》
4. 点击页面中央绿色的"登录"按钮进入下一步
5. 如需使用其他号码，可点击下方的"使用其他手机号登录"进行更换

## 第二步：完善初始个人健康档案

登录成功后，系统会引导您进入"初始注册"页面，请在此完善您的基础身体数据：

1. **设置昵称** - 在顶部输入框填写您的昵称
2. **选择性别** - 点击人物图标，选择"男士"或"女士"
3. **设置出生年月** - 选择您的出生年份和月份
4. **输入身体指标** - 通过滑动刻度，分别选择您的身高（厘米）与体重（公斤）
5. **选择城市** - 设定您当前所在的城市
6. **授权微信运动** - 勾选确认您已阅读并同意隐私政策，并授权获取微信运动数据
7. **确认注册** - 点击绿色的"确认注册"按钮

## 第三步：注册系统识别 ID 与设置密码

1. **填写用户 ID** - 输入您自定义的 ID，这是系统识别您用户身份的专属凭证
2. **设置密码** - 输入密码并再次输入确认
3. **验证手机号** - 填写手机号码，点击"获取验证码"按钮，输入收到的短信验证码
4. **勾选协议** - 确认您已阅读并同意隐私政策
5. **提交信息** - 点击绿色的"确认"按钮提交信息

## 第四步：注册完成

完成所有步骤后，页面会跳转至庆祝画面，并提示"注册成功！"

点击屏幕中央绿色的"进入"按钮，即可正式进入柠健康 APP，开启您的专属健康管理体验。
