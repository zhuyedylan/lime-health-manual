# 运动区间个性化设置

为了让系统分析报告更精准地贴合您的个人体质，建议您进行专业的运动区间设置。

## 运动心率设置

进入心率设置页面，您可以：
1. 输入个人的"最大心率"（如175次/分）
2. 选择区间划分方式：
   - 最大心率固定百分比（默认）
   - 用户自定义心率区间百分比
3. 系统将心率科学地划分为：
   - 极限区
   - 无氧区
   - 有氧区
   - 燃脂区
   - 热身区
4. 点击右侧的画笔图标，可手动修改各个区间的百分比临界值

## 运动功率设置

进入功率设置页面，您可以：
1. 输入个人的"FTP（功能性阈值功率）"（如120瓦）
2. 选择划分方式：
   - 标准七分功率区间法（默认）
   - 用户自定义功率区间百分比
3. 系统涵盖了7个高阶功率区间：
   - 冲刺能力
   - 无氧能力
   - 最大摄氧能力
   - 乳酸阈值
   - 速度耐力
   - 基础耐力
   - 放松恢复
4. 同样支持点击画笔图标进行针对性的数值编辑
