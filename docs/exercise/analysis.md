# 锻炼详情与多维度数据分析

在动态列表中点击任意一条具体的锻炼记录，即可进入"锻炼详情"页面，查阅更专业的运动分析。

## 核心数据看板
详情页顶部展示了完整的运动轨迹、天气和描述。

下方罗列了13项专业数据指标：
- 运动时间
- 总距离
- 卡路里
- 碳减排
- 运动强度
- 训练负荷
- 平均心率
- 平均配速
- 平均步频
- 累计爬升
- 最大心率

## 分时与分段数据分析

点击绿色的"分时锻炼数据"按钮，可进入深度图表页。

### 指标统计与区间
该页面自上而下展示了海拔、速度、心率、踏频和功率的平均值与最大值波动图，底部详细列出：
- "心率区间"（如燃脂、有氧等5个区间）
- "预估的功率区间"（7个区间）

在本次运动中的时间占比。

### 多指标交叉对比图表
点击特定指标卡片，可进入横屏交互式图表页面。您可以自由组合查看：
- "海拔+速度"
- "海拔+功率"
- "海拔+功率+速度"的叠加曲线

图表右下角支持在"时间"和"距离"两个横坐标维度间自由切换。

## 生理负荷和强度日历
在详情页底部的"生理负荷和强度"标签中，系统提取了锻炼时长、卡路里、锻炼距离、平均心率、最大摄氧量、有氧恢复、静息卡路里和训练负荷等指标，以微型折线图的形式，直观展示您近期的身体状态和表现趋势。
