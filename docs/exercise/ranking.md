# 多维度锻炼排行榜

锻炼板块内置了丰富的排行榜系统，激发您的运动动力。

## 排行榜分类
排行榜分为"步数"、"动态"、"俱乐部"和"挑战"四大统计维度。

## 排行榜筛选
排行榜下设"好友榜"、"地区榜"和"俱乐部"榜单。

您可以通过顶部的下拉菜单，筛选全国或特定的公司/组织，并按时间周期（如近一年）、年龄层和性别进行精准过滤。

## 动态排行榜聚合
在"动态"分类下，页面会展示当前筛选条件下的：
- 总参与人数
- 总次数
- 总距离
- 碳减排等聚合数据

您可以在"总公里"、"总时长"和"总次数"三个维度间切换排名依据。

## 榜单互动
排行榜前三名会以高亮的领奖台形式展示。

对于列表中的其他运动达人，您可以直接点击右侧的按钮进行"添加"或查看"已关注"状态，轻松扩展您的运动社交圈。
