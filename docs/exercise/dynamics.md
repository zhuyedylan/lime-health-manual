# 锻炼板块 - 动态展示与社交互动

锻炼主页上方设有"动态"、"我"和"俱乐部"三个标签页，为您打造瀑布式的运动社交体验。

## 动态瀑布流
在"动态"页中，您可以：
- 查看自己和所有好友的运动记录列表
- 每条动态卡片展示运动轨迹地图、天气、运动感言
- 核心数据：运动时间、总距离、卡路里消耗和碳减排量
- 对好友的动态进行点赞、评论和分享

## 快捷加入活动
动态流中会智能穿插推荐相关的"俱乐部"、"线上比赛"和"挑战"卡片，您可以点击卡片上的绿色"加入"按钮快速参与。

## 俱乐部动态筛选
切换至"俱乐部"标签，您可以在顶部下拉菜单中选择您加入的特定俱乐部（如高尔夫俱乐部）。

系统提供了强大的筛选功能，支持：
- 时间（如近一年）
- 时间升降序
- 运动项目（全部锻炼）
- 年龄和性别

页面上方会汇总展示该俱乐部在筛选条件下的总距离、总时间、总次数等，并标明您的个人排名情况。
