# 挑战板块指南

通过丰富的激励机制和社交互动，帮助您更好地坚持运动与健康习惯。

## 挑战的分类与统计逻辑

### 挑战发起主体
- 柠健康（系统管理员）
- 俱乐部管理员
- 普通用户自由发起

### 三大挑战类型
- **习惯类**：步行步数、睡眠时长、喝水量等日常习惯指标
- **健康类**：体重、心率、血压、体脂率等身体健康数据
- **运动类**：单车、跑步、游泳、健身等各类运动项目

### 三种统计逻辑
- **频次挑战**：设定特定日期内达到某项目标值的次数（例如：单次达标步数5000步，共需完成20天）
- **累计挑战**：在特定时间内，累计完成的总目标值（如总距离、总时长）
- **竞技挑战**：特定一级运动项目的专业挑战（如游泳的划水次数、SWOLF等）
