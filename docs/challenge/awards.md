# 个人挑战管理与领奖

点击挑战主页右上角的奖杯图标，可进入"我的挑战"综合管理页面。

这里包含五个标签栏：
- 我发起
- 进行中
- 已结束
- 已获奖
- 推荐中

## 挑战管理
在"我发起"列表中，您可以对发起的挑战点击"管理"按钮进行维护。

在"已结束"列表中，您可以查看挑战的最终结算状态。

## 名单下载
作为发起人或管理员，您可以在"挑战获奖名单"页面查看所有达标用户，并点击"保存获奖名单电子表格"将其导出。

## 领取奖品
在"已获奖"列表中，成功完赛的挑战会显示黄色的"去领奖"按钮。

点击后，系统会为您颁发精美的电子"证书"。点击"保存证书并领取奖品"后，需在"领奖信息确认"页面核实您的头像、昵称、电话、姓名、身份证信息及收件地址，确认无误后提交即可等待奖品发放。奖品发放后，按钮状态会更新为"已发放"。
