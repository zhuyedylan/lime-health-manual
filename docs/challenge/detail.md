# 挑战详情与排行榜

点击任意一个挑战卡片，即可进入"挑战详情"页面。

## 挑战要求与描述
页面顶部会详尽展示：
- 发起人
- 参与人数
- 活动时间
- 明确的挑战达标条件（例如："每日步行步数达到5000步的天数超过20天"）
- 详尽的挑战描述

## 挑战奖品
如果该挑战设有奖励，您可以在详情页查看奖品图片、赞助商信息以及获奖规则（如"所有完成挑战的都发放"）。

## 挑战排行榜
详情页下方设有实时更新的排行榜。您可以：
- 查看前三名的"领奖台"展示
- 查看其他参与者的完成次数排名
- 通过点击"关注"按钮关注优秀的运动达人

详情页底部提供"退出挑战"、"邀请好友"等快捷操作选项。
