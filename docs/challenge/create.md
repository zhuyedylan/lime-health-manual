# 如何发起挑战

如果您想作为发起人组织一场挑战，请点击挑战主页顶部的"+"号进入"发起挑战"页面。

## 基础信息填写
1. 输入您的"挑战标题"（必填）
2. 输入"挑战描述"
3. 从系统图库中选择一张合适的"挑战主图"

## 设定时间与内容
1. 选择挑战的时段
2. 从"习惯类、健康类、运动类"中选择一项挑战指标
3. 选择"频次挑战、累计挑战或竞技挑战"模式
4. 设定具体的"目标值"和"频次目标"
5. 页面中间会实时生成"预览挑战内容"的规则文案

## 设置奖励与收费（可选）

### 开启挑战奖励
您可以设置具体的奖品名称、赞助商、发放方式（如电子奖品）等。

### 开启付费挑战和奖金池
您可以设定奖金池比例（如报名费用的80%进入奖金池），并选择分配规则（如"所有完成挑战的均分"）以及填写收费用途描述。

## 发布
勾选同意服务协议后，点击底部的"发起挑战"即可发布。
