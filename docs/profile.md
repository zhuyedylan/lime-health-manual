# 个人资料管理

## 主页名片与专属二维码

在"我的"页面最上方，直观展示了您的头像、昵称、关注人数、粉丝数以及累计锻炼次数。

### 专属二维码海报
点击名片右上角的二维码图标，系统会为您生成一张专属的"用户二维码"海报，海报上包含：
- 供好友扫码添加您的二维码
- 您加入柠健康的日子
- 各项主要运动（游泳、跑步、骑行、健步走）的累计总时长与距离

## 个人展示页面

点击您的头像或名片区域，即可进入对外的"个人页面"。

在这个页面中，您可以预览他人视角的展示效果（他人可在此点击"关注"或发送"私信"）。

页面向下依次汇总了您的核心：
- **运动档案** - 总公里、总分钟、总次数
- **运动等级徽章** - 解锁的等级徽章
- **比赛奖牌** - 获得的比赛奖牌
- **所在俱乐部** - 加入的社群组织
- **我的挑战** - 参与的挑战活动
- **全部动态** - 瀑布流形式展示

## 基本信息完善

在基本信息设置页面，您可以深度完善您的个人档案：

- 修改头像、昵称、个性签名
- 修改性别、微信号、所在地区和国家
- 绑定真实姓名、生日、身份证及其他证件信息
- 维护实体收件地址
