# 习惯主页与日常概览

## 进入板块
在APP底部导航栏点击"习惯"图标，即可进入健康习惯主页。

## 今日进度提醒
页面最上方会清晰展示您"今日已完成"的习惯数量及对应的彩色图标。

系统会根据您的完成情况提供反馈：
- 未完成目标时提示"尚未完成的目标，请加油！"
- 全部达成时显示"完美的一天，全部目标已完成！"的庆祝卡片

## 习惯卡片看板
主页以卡片列表的形式展示您关注的各项健康指标（例如：步数、卡路里、睡眠、喝水、游泳等）。

## 快速查看进度
每张习惯卡片标明了您的每日目标（如"每日3000步"、"每日800卡路里"），还直观展示了从周日到周六每一天的完成情况打卡圈。

卡片底部提供本周的快捷统计：
- 本周完成天数
- 目标完成度百分比
- 本周累计总数
- 每日平均值
