# 健康习惯: 详细分析与数据评估

点击主页任意一个习惯卡片，即可进入详细的数据显示分析页面。

## 时间维度视图

### 周视图（本周统计）
页面上方提供当周数据的柱状图，直观展示每日完成量与目标线的对比。

"本周统计数据"包含：
- 总计完成量
- 达标天数
- 相关附加数据（步数对应的总距离、楼层数、消耗卡路里等）
- 每周平均值（与上周数据进行环比对比）
- 最好和最差的月份

底部"达标详情"展示当周每一天是否达成的具体记录。

### 月视图（本月统计）
柱状图扩展为显示整月每一天的数据起伏。

"本月统计数据"提供：
- 当月的总计数据及平均值
- 与上月平均值对比

底部的"达标详情"以日历的形式展现，盖有圆圈标记的日期即代表当天成功达标。

### 年视图（本年统计）
图表按月份展示全年的数据柱状图。

系统将统计您全年的累计数据及达标天数，并与去年的表现进行对比。

年度"达标详情"采用色块热力图展示，颜色从浅至深代表达标率的区间：
- <20%
- 20-40%
- >80%

在任何视图中，您都可以随时点击下方的蓝色按钮（如"返回本周"、"返回本月"、"返回本年"）快速回到当前时间周期。
