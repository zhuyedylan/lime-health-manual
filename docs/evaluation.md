# 健康评估

![柠健康值](/img/1柠健康值-1.png)

通过多维度的数据采集与深度分析，为您全方位的身体健康状态评估。

## 综合健康值概览

健康评估的最高等级即为"健康值"，是您整体健康状况的直观体现。

### 健康值雷达图
页面中央的雷达图展示了您当天的核心健康得分，五个维度：
- 活跃
- 体征
- 运动
- 睡眠
- 心脏

### 分值动态变化
分数下方会提示您今日得分与昨日的对比情况（如"比昨天提升了15分"）。

## 九大类健康指标得分

综合健康值由9个大类指标得分计算而成：

| 大类 | 说明 |
|------|------|
| Activity 活力 | 日常活动量 |
| Sleep 睡眠 | 睡眠质量和时长 |
| Figure 体态 | 身体形态指标 |
| Exercise 锻炼 | 运动表现 |
| Heart 心血 | 心血管健康 |
| Stress 压力 | 压力水平 |
| Recovery 恢复 | 身体恢复能力 |
| Breathe 呼吸 | 呼吸功能 |
| Healthcontrol 健控 | 健康管理能力 |

## 趋势分析与改善行动

### 专项趋势分析
对比本周与上周的数据，得出您的表现是上升还是下降的结论。

### 指标智解百科
- 指标详解 - 医学或健康学定义
- 问题危害 - 不达标的影响
- 改善措施 - 切实可行的调整方案

### 指标改善与挑战
系统会根据您的弱项指标，推荐相应的趣味运动挑战。
