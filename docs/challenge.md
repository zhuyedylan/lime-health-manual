# 挑战板块

![挑战](/img/3挑战-1.png)

通过丰富的激励机制和社交互动，帮助您更好地坚持运动与健康习惯。

## 挑战分类

### 发起主体
- 柠健康（系统管理员）
- 俱乐部管理员
- 普通用户自由发起

### 三大挑战类型
- **习惯类** - 步行步数、睡眠时长、喝水量等
- **健康类** - 体重、心率、血压、体脂率等
- **运动类** - 单车、跑步、游泳、健身等

### 三种统计逻辑
- **频次挑战** - 在特定日期内达到目标值的次数
- **累计挑战** - 在特定时间内累计完成的总目标值
- **竞技挑战** - 特定运动项目的专业挑战

## 挑战主页与日常管理

### 我参加的挑战进度
集中展示了您当前正在参与的所有挑战卡片。卡片上通过进度条、已完成次数和"剩余天数"直观显示您的当前进度。

## 加入挑战

浏览列表时，遇到感兴趣的活动点击右侧的"加入挑战"按钮即可参与，已加入的挑战会显示为"已加入,查看"。

## 发起挑战

您也可以作为发起人组织一场挑战：

1. 输入挑战标题和描述
2. 选择挑战类型和指标
3. 设定目标值和频次目标
4. 设置挑战奖品（可选）
5. 点击"发起挑战"发布

## 邀请与分享挑战

在挑战详情页点击邀请后，系统会生成专属的挑战海报，包含活动的二维码或小程序码。

您可以选择：
- 保存图片并分享
- 在APP里面分享
- 微信里面分享

## 个人挑战管理与领奖

点击挑战主页右上角的奖杯图标，可进入"我的挑战"综合管理页面。

包含五个标签栏：
- 我发起
- 进行中
- 已结束
- 已获奖
- 推荐中

成功完赛的挑战会显示黄色的"去领奖"按钮，领取精美电子"证书"。
