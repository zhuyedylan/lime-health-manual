# AI积点管理与充值购买

每次请求AI助手启动分析，都需要消耗一定的"AI积点"。

系统会根据您开通的不同会员类型定期赠送一定数量的积点，如果您积点额度不足，可以直接进行充值购买。

## 积点查询与收支明细
在"AI积点管理"页面，您可以清晰地看到您账户中的"当前可用积点"和"本月消费"总数。

页面下方提供了历史明细列表，支持按时间范围筛选，帮助您追溯每一笔积点的消耗去向（如生成一份一级健康指标分析扣除10点）以及充值记录。
