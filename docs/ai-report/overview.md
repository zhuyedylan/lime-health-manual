# AI报告与分析板块

该板块基于您日常记录的各项健康、身体与运动数据，通过人工智能技术为您生成深度的解读报告和个性化的改善计划。

在"我的"页面上方点击您的AI点数图标或报告入口，即可进入该功能模块。

## AI报告类别与启动分析

进入"AI运动健康报告"主页，页面上方展示了系统支持生成的六大类深度分析报告：
1. 健康值报告
2. 一级健康指标报告
3. 二级健康指标报告
4. 运动报告
5. 综合报告
6. 年度报告

页面下方会集中展示您最近生成的各类AI报告历史记录，未查看的报告会标有绿色的"未读"提示。

## 启动分析步骤
1. 点击进入您需要的特定报告类型页面（如"AI健康值报告"或"AI运动报告"）
2. 通过下拉菜单选择您想要分析的具体时间范围、特定日期，或某一特定的锻炼动态
3. 对于层级指标报告，您还需要选择具体的指标大类（如"睡眠"、"活力"）
4. 选定后，点击页面中央蓝色的"启动分析"按钮
5. 系统弹出AI助手生成进度条，完成后即可自动跳转至详细的报告页面
