# AI一、二级健康指标报告

当您需要针对健康值中的某项短板进行深入研究时，可以使用这两类报告。

## 一级健康指标报告详情

对活力、睡眠等大类进行拆解。报告会给出该大类的综合评级（如C+），并将构成该大类的所有下属二级指标（如步数、活动卡路里、静息心率等）的图表数据汇总展示。

报告重点提供：
- 一级指标分析
- 二级指标关联分析

帮助您找出影响该大项得分的核心原因，并生成周级别的具体改进目标。

## 二级健康指标报告详情

聚焦单一维度的精准打击（如针对"睡眠"中的"深睡比例"或"血氧"）。

报告不仅提供您的实际指标走势，还会进行深度的"二级指标经验比较"，对比您的数据与同龄人的健康基准值差距，解析该单一问题对您整体健康评级的影响，最后为您提供改善该具体问题的针对性建议措施（如调整固定入睡时间）。
