# AI运动报告详情

该报告是您专属的私人运动教练复盘。

针对您选择的某一次锻炼（如单车或游泳），报告不仅提供基础的得分、运动类型归类和所需的恢复时间（如18-24小时），更会结合您的运动轨迹图和基础消耗数据，分模块为您输出专业的点评。

## 分析内容

### 运动描述
记录本次锻炼的基本信息。

### 运动能力
分析如SWOLF划水效率，并指出配速稳定性的不足。

### 运动与心肺
评估心肺功能表现。

### 运动与能耗
分析能量消耗情况。

### 运动与健康关联分析
评估本次锻炼对整体健康的影响。

## 下周计划
在页面的最后，AI会基于本次运动表现，为您全盘规划一份极致详细的"下周锻炼与饮食计划"，细化到周一至周日每一天的训练项目（如核心训练、有氧恢复）以及包含高蛋白早餐与低脂午餐在内的针对性饮食方案。
