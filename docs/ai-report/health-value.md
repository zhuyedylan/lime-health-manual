# AI健康值报告

该报告直观反映您的整体健康状态。

## 报告内容
- 页面顶部给出当天的综合得分（如76分）
- 用曲线展示近期的分数波动趋势以及最低点提示
- 通过"蛛网模型关联性"全面评估九大维度：
  - Activity 活力
  - Sleep 睡眠
  - Heart 心血
  - 等...

- 附带"指标等级分布表"汇总您各项维度的A级天数表现
- 自动为您揪出薄弱指标（如"Recovery恢复"被评C级或D级）
- 为您的"下周具体改进措施"，例如增加日光暴露时间或延长正念练习

## 指标等级说明
- A级：表现优秀
- B+ / B / B-：表现良好
- C+ / C / C-：需要关注
- D+ / D / D-：需要改善
- E+ / E / E-：需要重点关注
