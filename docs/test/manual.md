# 手动打卡和测试板块

当您有未被智能设备自动记录的健康数据，或需要进行专业的自我健康状态评估时，可以通过此板块进行手动补充。

在"评估"主页底部点击"手动打卡和测试"按钮即可进入该页面，板块主要分为"手动打卡"与"测试"两大类。

## 一、手动打卡（日常指标录入）

该区域允许您快捷录入13项日常身体与运动指标。

### 支持记录的指标
- 睡眠时间
- 步数
- 锻炼
- 体温
- 身高
- 体重
- 腰围
- 臀围
- 血压
- 心率
- 血氧
- 静心心率
- 血糖

### 通用打卡步骤
1. 选择时间：在顶部下拉菜单中选择您要记录数据的具体"日期"与"时间"
2. 录入数值：页面中央设有便捷的滑动刻度尺，左右滑动即可精确选择您的指标数据
3. 提交与查询：点击绿色的打卡按钮完成提交，也可点击下方的"查看历史记录"按钮回顾该指标的过往数据

### 特殊指标录入说明

**锻炼打卡**：
- 需要先选择"锻炼类型"（如骑行）
- 手动输入"时长"、"距离"、"平均速度"、"起始时间"
- 可为本次锻炼添加自定义的"锻炼名称"

**血压打卡**：
- 需在两个独立的滑动刻度上，分别录入"高压"（收缩压）和"低压"（舒张压）

**血糖打卡**：
- 在录入具体数值前，需要先在数值上方选择您测量时的状态标签：
  - "空腹"
  - "餐后"
  - "随机"
