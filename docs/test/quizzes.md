# 测试（健康问卷评估）

该区域提供多维度的主观量表测试，帮助您深层评估自身的心理与生理负荷状态。

## 支持的测试类型
- 训练负荷
- 压力指数
- 心情状态
- 焦虑风险
- 抑郁风险
- 疲劳指数
- 综合健康值测试

## 完整测试流程（以"训练负荷测试"为例）

### 1. 阅读简介与开始
点击进入测试后，首屏会展示该测试的预计用时（如1分钟）、评估目的与原理说明及医学参考来源。点击底部的"开始测试"进入答题环节。

### 2. 进行答题
页面上方会有进度条显示您当前的答题进度。

请仔细阅读问题，并点击符合您近期真实情况的选项（如"完全没有"、"有些天"、"几乎每天"等）。

如果想修改上一题的答案，可点击底部的"回到上个问题"按钮。

### 3. 查看评估报告
完成所有题目后，系统会立刻生成结果页。

页面顶部会突出显示您的测试得分（如94分）及对应评级（如"极轻"），下方会附上针对您状态的详细解读与健康提示。

### 4. 复核与提交
在结果页向下滚动，您可以完整核对所有题目的作答情况。

确认无误后，点击底部的"确认提交"按钮进行保存；若觉得填写有误，可点击"取消重测"重新答题。

### 5. 最终确认
点击确认提交后，系统会弹出确认小窗口，汇总展示测试完成时间与得分，点击绿色"确认"键，该次测试数据即正式录入您的健康档案。
