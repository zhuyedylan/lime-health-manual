# 俱乐部挑战

在俱乐部的"挑战"专属页面中，集中展示了由该俱乐部官方或成员发起的各项挑战活动。

## 筛选与排序
您可以通过顶部的下拉菜单，切换查看不同俱乐部的挑战，还可以：
- 按时间范围（如"近一年"）进行筛选
- 按"time"、"升序"或"降序"重新排列挑战卡片

## 参与专属挑战
挑战卡片上清晰标明了：
- 发起人
- 挑战的主题（如"高尔夫夏季赛"、"骑行100公里去上海"）
- 活动时间
- 当前参与人数
- 涵盖的运动类型图标（如高尔夫、骑行、跑步等）

遇到感兴趣的挑战，点击右侧绿色的"加入挑战"按钮即可报名参与；对于您已经报名的项目，按钮会状态更新为"已加入,查看"。
