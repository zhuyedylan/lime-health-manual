# 俱乐部板块指南

俱乐部是供拥有相同运动爱好和健康目标的用户聚集、交流、分享与竞技的专属社群空间。

## 俱乐部首页与概览

### 浏览与加入
在俱乐部首页，您可以：
- 查看俱乐部的顶部海报图、专属Logo、名称及地理位置
- 阅读详尽的俱乐部图文介绍
- 点击页面中央绿色的"加入"按钮成为其中一员

### 基础数据看板
首页直观展示了该俱乐部的四项核心统计数据：
- 当前的会员人数
- 活动总数
- 挑战数量
- 发表的文章总数

下方还配有近期活跃成员的头像展示。

### 数据排行为与个人对比
在"本月俱乐部数据和排行榜"模块中，系统会统计俱乐部本月的总公里数和总运动时长。

系统会自动将您的个人运动数据与俱乐部的平均水平进行对比，为您提示：
- "比人均活动次数多多少次"
- "比人均运动距离多多少公里"
- "比人均运动时间多多少小时"

点击右侧的"查看俱乐部排行榜 >"可进入更详细的榜单页面。
