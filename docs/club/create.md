# 俱乐部文章创作与编辑

如果您有分享心得的需求，可以通过首页顶部的"+"号进入新建文章页面。

## 设置基础信息
1. 输入限制在30字以内的文章简介（标题）
2. 点击相机图标上传一张文章主图

## 多模块内容排版
文章编辑器提供了高度自由的模块化排版功能。

您可以通过页面底部的工具栏，灵活添加：
- 大图
- 小图（支持多图网格拼贴）
- 文字段落
- 视频

在编辑过程中，每两个模块之间还会出现"添加内容"的快捷按钮，方便您随时插入新素材。

## 排版调整与发布
对于已添加的各个内容模块，您可以：
- 选中后点击"上移"、"下移"或"删除"进行重新排序和调整
- 左下角支持一键撤销（向左箭头）和重做操作

确认排版和内容无误后，点击页面底部绿色的"发表文章"或"完成编辑"按钮，即可将文章发布至俱乐部。
