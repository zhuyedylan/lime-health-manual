# 俱乐部文章阅读与互动

## 文章列表浏览
在"文章"标签下，您可以浏览俱乐部发布的最新资讯和动态。

文章缩略卡片上显示了：
- 发布时间
- 地点
- 文章标题
- 简短摘要

点击右上角的"阅读更多 >"或卡片本身即可进入全文。

## 多媒体阅读体验
文章详情页支持丰富的多媒体展示，包括多图并排（图集）展示以及嵌入式的视频播放器，点击播放按钮即可直接观看视频，带来沉浸式的阅读体验。

## 点赞与评论互动
在文章列表或详情页底部，您可以使用：
- 点赞（大拇指图标）
- 评论（气泡图标）
- 分享功能

在详情页的评论区，您可以查看所有成员的留言并进行互动。底部输入框支持输入文字、上传相册图片或直接拍照发送。

评论区支持多层级的楼层嵌套回复（如点击"展开11条回复"可查看子评论），方便成员间进行深度的交流与探讨。
