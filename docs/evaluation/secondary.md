# 二级指标详解与数据追踪

每个大类指标的最终得分均由其下属的若干核心二级指标共同决定。

点击任意一个二级指标（例如"睡眠时长"或"睡眠得分"），即可进入深度分析页面。

## 多维度时间视图
在详情页顶部，您可以自由切换"日"、"周"、"月"、"年"四个时间维度，系统会以直观的柱状图呈现您的历史数据与目标平均值。

## 指标结果与范围
图表下方会显示该指标的具体数值（如睡眠6小时），并辅以彩色刻度条，标明您的数据所处的健康范围。

## 评级情况
对于综合性的二级指标（如"睡眠得分"），页面会按天展示您的具体评级（从A+到E+），帮助您精确定位哪一天的表现存在不足。

## 相关指标一览
在页面底部，系统会将与该项紧密相关的其他二级指标（如睡眠周期、睡眠占比、睡眠心率、静心心率变异率、睡眠中断等）汇聚在一起，方便您进行横向关联比对。
