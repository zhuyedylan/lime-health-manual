# 九大类健康指标得分

综合健康值由9个大类指标得分计算而成，在雷达图下方，您可以看到这九大核心维度的卡片及其评级（如A、B+、C-等）。

## 九大类构成

系统将健康数据细分为以下九个大类：
1. **Activity 活力** - 日常活动量
2. **Sleep 睡眠** - 睡眠质量和时长
3. **Figure 体态** - 身体形态指标
4. **Exercise 锻炼** - 运动表现
5. **Heart 心血** - 心血管健康
6. **Stress 压力** - 压力水平
7. **Recovery 恢复** - 身体恢复能力
8. **Breathe 呼吸** - 呼吸功能
9. **Healthcontrol 健控** - 健康管理能力

## 快速概览
点击大类指标卡片右侧的箭头展开，即可看到该大类下包含的所有核心二级指标列表，每个二级指标右侧均附带微型趋势图，方便您一眼扫过近期的波动情况。

## 今日建议
在九大类指标列表下方，系统会结合您各维度的具体表现（特别是评级较低的B、C项），生成一段详细的"今日建议"。建议涵盖作息、饮食、运动及心理调节等方面，为您提供个性化的健康指导。
