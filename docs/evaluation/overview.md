# 健康评估板块

在APP底部导航栏点击"评估"图标，即可进入健康评估主页。

## 综合健康值概览

健康评估的最高等级即为"健康值"，是您整体健康状况的直观体现。

### 日期与日历
页面顶部设有可滑动的日期栏，方便您随时查看过去每一天的健康评估结果。

### 健康值雷达图
页面中央的雷达图展示了您当天的核心健康得分（例如76分），五个维度：
- 活跃
- 体征
- 运动
- 睡眠
- 心脏

直观地反映了您在各项基础健康维度的均衡情况。

### 分值动态变化
分数下方会提示您今日得分与昨日的对比情况（如"比昨天提升了15分"），并给予简明扼要的鼓励或提醒。

### 手动打卡
如果您有未被系统自动捕获的健康数据，可点击页面下方的"手动打卡和测试"绿色按钮进行补充录入。
