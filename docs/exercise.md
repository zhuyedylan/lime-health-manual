# 锻炼功能

欢迎使用柠健康 APP 的"锻炼"功能板块。

该板块为您提供全面的运动数据追踪、深度分析、社交互动以及个性化参数配置。

## 动态展示与社交互动

锻炼主页上方设有"动态"、"我"和"俱乐部"三个标签页，为您打造瀑布式的运动社交体验。

### 动态瀑布流
在"动态"页中，您可以：
- 查看自己和所有好友的运动记录列表
- 每条动态卡片展示运动轨迹地图、天气、运动感言
- 核心数据：运动时间、总距离、卡路里消耗和碳减排量
- 对好友的动态进行点赞、评论和分享

### 快捷加入活动
动态流中会智能穿插推荐相关的"俱乐部"、"线上比赛"和"挑战"卡片。

## 锻炼详情与多维度数据分析

在动态列表中点击任意一条具体的锻炼记录，即可进入"锻炼详情"页面。

### 核心数据看板
下方罗列了13项专业数据指标：
- 运动时间、总距离、卡路里
- 运动强度、训练负荷
- 平均心率、平均配速、平均步频
- 累计爬升、最大心率

### 分时锻炼数据
点击绿色的"分时锻炼数据"按钮，可进入深度图表页，查看：
- 海拔、速度、心率、踏频和功率的波动图
- 心率区间（燃脂、有氧等5个区间）
- 功率区间（7个区间）

## 手动打卡和编辑锻炼

当您没有佩戴智能设备或需要补充记录时，可以使用"锻炼打卡和编辑"功能手动录入数据。

支持：
- 选择运动类型（骑行、跑步等）
- 输入时长、距离、平均速度
- 填写锻炼名称和心得
- 负荷自评 (RPE)

## 多维度锻炼排行榜

内置了丰富的排行榜系统，激发您的运动动力：

| 排行榜类型 | 说明 |
|-----------|------|
| 步数 | 日常步数排名 |
| 动态 | 动态互动排名 |
| 俱乐部 | 俱乐部内部排名 |
| 挑战 | 挑战完成度排名 |

支持筛选：好友榜、地区榜、俱乐部榜单
