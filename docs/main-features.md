# 我的板块

欢迎来到"我的"板块，这是您的个人专属综合管理中心。

在这里，您可以全面管理个人资料、查看健康与运动数据汇总、设置隐私权限以及使用丰富的快捷功能。

## 快捷入口

主页下方设有九宫格"快捷入口"，满足您快速触达核心功能的需求：

| 图标 | 功能 |
|------|------|
| 消息 | 集中处理所有互动与通知 |
| 收藏 | 分类管理您标记的重要内容 |
| 挑战 | 查看参与挑战的进度 |
| 俱乐部 | 浏览加入的社群组织 |
| 动态 | 检索历史运动记录 |
| 好友 | 管理关注列表 |
| 排行 | 查看各类排行榜 |
| 比赛 | 参与比赛活动 |
| 绑定 | 智能设备绑定 |
| 打卡 | 手动打卡 |

## 系统入口

点击"我的"页面右上角的齿轮图标，即可进入"我的设置"中心，进行系统设置与隐私安全管理。
