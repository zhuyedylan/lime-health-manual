# 手动打卡和测试

当您有未被智能设备自动记录的健康数据，或需要进行专业的自我健康状态评估时，可以通过此板块进行手动补充。

## 手动打卡

该区域允许您快捷录入13项日常身体与运动指标：

### 支持记录的指标
- 睡眠时间、步数、锻炼
- 体温、身高、体重
- 腰围、臀围
- 血压、心率、血氧
- 静心心率、血糖

### 通用打卡步骤
1. 选择日期与时间
2. 滑动刻度选择数值
3. 点击绿色打卡按钮提交

## 测试（健康问卷评估）

提供多维度的主观量表测试：

### 支持的测试类型
- 训练负荷、压力指数、心情状态
- 焦虑风险、抑郁风险、疲劳指数
- 综合健康值测试

### 完整测试流程
1. 阅读简介与开始
2. 进行答题（支持修改上一题）
3. 查看评估报告
4. 复核与提交
5. 最终确认

测试完成后系统会立刻生成结果页，显示测试得分及对应评级，并附上详细的解读与健康提示。
