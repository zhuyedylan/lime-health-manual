# 快捷入口

主页下方设有九宫格"快捷入口"，满足您快速触达核心功能的需求。

## 1. 我的消息

集中处理您的所有互动与通知，分为四个标签分类：

| 标签 | 内容 |
|------|------|
| 系统 | 锻炼导入提醒、挑战完成、比赛完赛、获得奖牌及等级提升等 |
| 私信 | 与好友的专属聊天对话窗口 |
| 评论和@我 | 好友在动态或文章中提及您的记录 |
| 赞 | 好友对您的动态点赞的明细 |

## 2. 我的收藏

分门别类地管理您标记的重要内容，包含四个分类标签：
- 动态
- 文章
- 挑战
- 比赛

## 3. 挑战与俱乐部追踪

- **我的挑战** - 直观查看您当前参与所有挑战的进度条及剩余天数
- **所在俱乐部** - 浏览您当前加入的所有社群组织列表

## 4. 用户动态检索

点击进入动态库后，可以通过顶部菜单进行交叉精确筛选：
- 时间范围（如近一年）
- 升序/降序
- 全部运动项目
- 全部设备

## 5. 其他入口

除上述功能外，快捷入口还包含：
- 我的好友
- 我的排行
- 我的比赛
- 智能绑定
- 手动打卡
