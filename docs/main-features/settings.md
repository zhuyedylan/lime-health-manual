# 系统设置与隐私安全

点击"我的"页面右上角的齿轮图标，即可进入"我的设置"中心。

## 身体数据精细管理

在"身体数据"菜单中，您可以精确记录和管理自身的各项生理指标：

### 身体参数
- 身高、体重、三围

### 体脂成分
- 体脂率、脂肪率、肌肉率、水分率

### 心血管代谢数据
- 步行平均心率、静息心率、基础代谢

### 核心健康指标
- 血压、血氧、血糖

## 安全隐私配置

在"隐私设置"中，您可以：
- 设置"个人运动动态"、"个人资料页"、"关注列表和粉丝列表"的可见范围
- 控制"是否需要关注好友申请"
- 控制"是否展示我可能感兴趣的内容"
- 控制"是否无限接受未关注对象的消息"

### 系统授权管理
- 手机位置信息
- 相机
- 相册
- 麦克风
- 本地网络
- 健康数据

## 账户管理与客服联络

如遇问题，可点击"联络客服"，获取官方客服电话（16816888888）及客服专属微信二维码。

设置中心还支持：
- 常用装备管理
- 运动数据同步
- 智能设备绑定
- 急救设定
- 退出账户
- 注销账户
