# 会员与权益

## 一、购买与订阅柠健康会员

### 1. 选择会员类型
进入"柠健康会员中心"页面后，您可以自由切换"金卡"和"铂金卡"两个标签，查看不同等级的会员权益。

### 2. 选择订阅套餐
系统为您提供了多种灵活且优惠的订阅方案：
- 连续包月（首月11元，次月16元）
- 连续包季（32元）
- 连续包年（130元）

### 3. 确认与支付
在页面下方勾选确认圆圈，表示您已阅读并同意相关协议。点击蓝色的"确认协议，去开通会员"按钮，完成支付即可立即激活相关权益。

## 二、激活合作尊贵会员

### 合作伙伴列表
- 国寿财险客户免费开通
- SETC铁三俱乐部 599元

### 激活方式 A（使用序列号）
1. 在"激活尊贵会员"页面输入合作伙伴提供的"权益序列号"
2. 填写本人的手机号码，获取并输入短信验证码

### 激活方式 B（使用手机号）
1. 直接输入"您在合作伙伴注册的手机号"
2. 获取并输入验证码进行身份校验

### 完成激活
仔细阅读相关服务及隐私协议并勾选确认后，点击绿色的"激活权益"按钮，即可成功绑定该尊贵会员卡。

## 三、我的会员管理中心

### 查看已有会员
进入"我的会员"页面，可以看到您当前拥有的所有会员卡片：
- 金黄色的"柠健康会员连续包月卡"
- 银灰色的"合作尊贵会员卡"

卡片右下角会清晰标明该服务的具体到期日期。

## 四、核心会员权益详解

### 1. 智能锻炼与运动特权
- 专属的智能运动计划（跑步、瑜伽、力量训练）
- 虚拟AI教练提供详细指导视频、动作演示及实时训练反馈
- 热门运动场馆预订优惠
- 专属商城中的优惠运动商品购买特权

### 2. 运动风险与安全守护
- AI健康顾问实时评估运动风险
- 高强度训练时提供疲劳预警和安全建议
- 监测到过度运动时自动提醒休息
- 运动后的科学恢复建议

### 3. 全方位健康评估与身体趋势追踪
- 定期的深度健康评估服务
- 长期的身体趋势数据追踪视图
- AI自动分析健康进展

### 4. AI健康顾问专属服务
- 会员专享私人AI健康顾问互动
- 通过语音或文字随时咨询
- 提供涵盖饮食、作息、心理等方面的定制化健康计划

### 5. 习惯打卡与健康挑战
- 支持每日记录饮水、运动、睡眠等习惯
- 个性化打卡提醒，完成任务后获得积分奖励
- 专属健康挑战活动（步数、时长、睡眠等）
- 赢取专属徽章、积分或优惠券奖励

### 6. 低碳积分计划
- 日常步行、骑行等绿色出行活动转化为"低碳积分"
- 积分可兑换健康产品、专属折扣或服务升级

### 7. 高级俱乐部会员权益
- 优先体验最新健康挑战和智能锻炼计划
- 解锁专属的健康报告专家级解读服务
- 线上专家咨询
- 参与定期举办的线下健康活动与运动会
