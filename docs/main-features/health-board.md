# 健康看板与会员权益

## 核心健康状态速览

在个人名片下方，系统为您提供直观的健康状态看板。

### 身体电量趋势图
查看近期的精力波动百分比。

### 运动负荷趋势图
-"本周趋势-运动负荷"图表专业呈现：
- **长期负荷（CTL）** - 长期训练负荷
- **短期负荷（ATL）** - 短期训练负荷
- **训练疲劳度（TSB）** - 训练疲劳度曲线

## AI报告与商城

看板下方设有"身体报告"入口，您可以：
- 通过 AI 深度解析您的健康数据
- 进入会员商城
- 进入风险保障页面

## 柠健康会员中心

点击会员卡区域可进入会员中心。

### 会员等级
- **金卡会员** - 基础高级权益
- **铂金卡会员** - 尊享更多权益

### 订阅方案
- 连续包月（首月11元，次月16元）
- 连续包季（32元）
- 连续包年（130元）

### 合作尊享会员
- 国寿财险客户免费开通
- SETC铁三俱乐部 599元
